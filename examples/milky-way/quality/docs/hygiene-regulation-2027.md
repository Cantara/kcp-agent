# 2027 hygiene regulation — gap analysis

Enters into force 2027-01-01. Main changes for dairy plants:

- Allergen changeover validation moves from annual to quarterly.
- Environmental monitoring extends to drains and conveyor undersides.
- Digital CCP records become mandatory (paper logs no longer accepted).

Gap: CCP-2 chamber sensors already comply; CCP-3 swab cadence must double.
Budget request filed for Q4 2026.
