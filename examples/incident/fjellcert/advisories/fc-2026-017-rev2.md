# FC-2026-017 rev2 — Quaymaster Broker: patch available

*(Fictional example advisory. Supersedes FC-2026-017.)*

**Update:** Quaymaster Systems has released Broker 4.8.2, which closes the
remote code execution vulnerability in the management listener.

**Action:** upgrade all brokers to 4.8.2. After upgrading, rotate broker
credentials and re-enable the management listener only where required.
The interim listener-off mitigation is no longer sufficient as a long-term
posture: exploitation attempts against re-enabled listeners continue.
